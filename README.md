# aliyundriver-webdav 

#### 介绍
{**阿里云盘/本地挂载网络盘/WebDAV/win软件/界面**}

https:fyne.io/fyne/v2 v2.1.2 ui框架

https:github.com/getlantern/systray 任务栏(win10 bug不少 弃用)
